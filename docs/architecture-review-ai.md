# Architecture review AI

Developed by the LetsCookTech Open Source Team.

Architecture review AI helps teams evaluate system boundaries, data flow, dependencies, scalability, reliability, cost, security, deployment, rollback, and maintainability before mistakes become expensive.

## Difference from generic AI advice

Generic AI often recommends patterns. Arsenal review requires trade-offs, failure modes, verification, ownership, and a decision record.
