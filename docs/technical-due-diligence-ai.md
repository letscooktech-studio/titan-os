# Technical due diligence AI

Developed by the LetsCookTech Open Source Team.

Technical due diligence AI reviews a product or codebase for risks that affect acquisition, investment, launch readiness, security posture, scalability, maintainability, team execution, and cost exposure.

## Due diligence checklist

- architecture risks;
- security risks;
- database and data risks;
- operational maturity;
- technical debt;
- cost exposure;
- roadmap credibility;
- AI system reliability.

Start with [`cto-operating-system`](../skills/cto-operating-system), [`security-auditor`](../skills/security-auditor), and [`technical-debt-hunter`](../skills/technical-debt-hunter).
