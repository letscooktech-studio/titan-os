# Case studies

Developed by the LetsCookTech Open Source Team.

## Research agent

Expected catches: source reliability, citation drift, browser-tool injection, memory retention, and unsupported claims.

## Coding agent

Expected catches: destructive file operations, test bypass, dependency risk, secret exposure, and weak rollback.

## Browser agent

Expected catches: credential handling, page injection, task-boundary confusion, and unsafe form submission.

## Multi-agent system

Expected catches: duplicate work, conflicting objectives, deadlocks, shared memory poisoning, and runaway cost.
