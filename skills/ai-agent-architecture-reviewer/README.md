# AI Agent Architecture Reviewer

Developed by the LetsCookTech Open Source Team.

A production-grade review framework for AI agent systems: single agents, multi-agent systems, copilots, workflow agents, browser agents, coding agents, research agents, and operations agents.

## What it catches

- planning loops and decomposition failures
- memory drift, poisoning, context explosion, and privacy risk
- tool hallucination, tool abuse, and permission gaps
- MCP and external API security issues
- prompt injection and indirect prompt injection
- weak verification and unsupported claims
- cost fan-out and uncontrolled retries
- observability gaps and black-box execution

## Try it

```text
Use $ai-agent-architecture-reviewer to review this AI agent before deployment.
Artifacts: architecture diagram, tool list, permissions, prompts, memory design, logs, evals, deployment plan, and cost assumptions.
Return evidence-backed risks, mitigations, verification tests, and production-readiness score.
```
