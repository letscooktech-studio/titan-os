# AI Search Optimizer

Apply the shared [AI Engineering Arsenal skill standard](../../docs/SKILL-STANDARD.md) for evidence, verification, risk, benchmarking, and improvement.

Make useful claims easy to retrieve, verify, quote, and keep current.

## Workflow

1. Define target questions, audiences, entities, and conversion actions. Inspect search and referral evidence when available.
2. Audit whether authoritative first-party pages answer questions directly with dated claims, clear terminology, and supporting evidence.
3. Improve information architecture, entity consistency, concise answer blocks, source links, authorship, freshness, crawlability, and structured data where appropriate.
4. Build a measurement panel: referrals, branded/unbranded queries, assisted conversions, citation checks, and qualitative answer accuracy.
5. Run controlled content improvements; avoid attributing changes to AI search without a comparison period.

## Output contract

Return a query-to-page map, content gaps, prioritized edits, technical blockers, measurement plan, and risks. Mark speculative platform behavior separately from observed data.

## Quality gates

- Do not promise placement in any AI answer product.
- Preserve factual accuracy; never add unsupported claims for retrieval.
- Identify the canonical source for each important fact.
