# Benchmark

Developed by the LetsCookTech Open Source Team.

## Required comparison

Run the same Next.js project through:

1. default AI review;
2. framework review using `SKILL.md`;
3. evaluator scoring using `RUBRIC.md`.

## Reports to publish

- default AI result;
- framework result;
- coverage comparison;
- risk detection comparison;
- SEO and AI-search comparison;
- actionability comparison;
- accuracy review;
- missed issues report;
- time-saved estimate or measurement.

Do not claim superiority until raw outputs and scoring are published.
