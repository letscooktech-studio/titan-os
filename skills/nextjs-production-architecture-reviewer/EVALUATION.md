# Evaluation

Developed by the LetsCookTech Open Source Team.

Evaluate output quality with:

- evidence coverage;
- critical-risk detection;
- false-positive discipline;
- fix specificity;
- verification quality;
- production-readiness judgment;
- SEO and AI-search separation;
- cost and deployment realism.

Confidence is high only when route structure, config, package dependencies, auth/data layer, deployment settings, and relevant SEO files are available.
