# Supabase Production Auditor

Developed by the LetsCookTech Open Source Team.

Production-grade Supabase auditing for SaaS, marketplaces, AI apps, mobile apps, ecommerce, learning platforms, and internal tools.

## What it catches

- missing or weak RLS
- cross-tenant access
- service-role misuse
- public storage exposure
- insecure auth and admin flows
- slow queries and missing indexes
- Realtime fan-out and bandwidth risks
- backup, migration, and incident gaps
- Supabase cost explosion risks

## Try it

```text
Use $supabase-production-auditor to audit this Supabase project before launch.
Artifacts: schema SQL, migrations, policies, storage bucket settings, Edge Functions, auth flows, and app Supabase client usage.
Return evidence-backed risks, fixes, verification queries, and production-readiness score.
```
