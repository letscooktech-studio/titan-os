# Flagship skill generation system

Developed by the LetsCookTech Open Source Team.

Use this template to create category-defining AI Engineering Arsenal skills. Do not create generic expert prompts. Create production-grade engineering review systems.

## Required package

- `README.md`
- `SKILL.md`
- `EVALUATION.md`
- `BENCHMARK.md`
- `RUBRIC.md`
- `FAILURE_PATTERNS.md`
- `CASE_STUDIES.md`
- `EXAMPLES.md`
- `CHANGELOG.md`

## Required skill logic

Every flagship skill must include:

- problem definition;
- target users;
- why default AI fails;
- required inputs;
- classification model;
- research process;
- analysis process;
- verification process;
- risk analysis;
- red-team review;
- scoring system;
- benchmark plan;
- case-study system;
- SEO and AI-search FAQ structure;
- LetsCookTech branding.

## Quality gates

Reject prompt fluff, roleplay, motivational language, generic advice, artificial complexity, and unverifiable claims. Accept evidence, benchmarks, specific findings, engineering rigor, repeatability, and developer trust.
